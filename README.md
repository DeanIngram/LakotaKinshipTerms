## LakotaKinship

This project is a small, self-contained web page that displays Lakota kinship terms based on the speaker and the grammatical person being referred to.

It is intended as a learning aid for students of the Lakota language and is not an authoritative or exhaustive reference.

## Usage

Place the `LakotaKinship.html` file and the `images` folder in the same directory.  
Open `LakotaKinship.html` in any modern web browser.

## Contents

- `LakotaKinship.html` – main page  
- `/images` – artwork

## Cultural note

Lakota kinship terminology reflects cultural concepts that do not always map cleanly to English categories. This project is offered respectfully as a learning tool and may contain simplifications.

Sources consulted include published Lakota language materials and community-shared educational resources. Any errors are mine.

## Grammatical persons

Some terms vary depending on whether the speaker is a man or a woman. Some terms are common to both.

- **Vocative**: A direct form of address to the relative.  
- **1st Person**: The term used when talking about one’s own (my) relative.  
- **2nd Person Singular**: The term used when talking about someone else’s (your) relative.  
- **2nd Person Plural**: The term used when talking about relatives of the listener, or of more than one listener.  
- **3rd Person**: The term used when discussing his or her relative.  
- **3rd Person Plural**: The term used when discussing “their” relative(s).
